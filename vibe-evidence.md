# Minh Chứng Dùng AI Để Sinh Giao Diện

File này ghi lại các bằng chứng về việc nhóm sử dụng AI (vibe coding / AI-assisted design) để xây dựng giao diện của ScamChecker.

---

## Công cụ AI đã dùng

| Công cụ | Mục đích |
|---------|---------|
| Replit Agent | Sinh toàn bộ cấu trúc dự án, HTML/CSS/JS, cấu trúc thư mục |
| ChatGPT / Claude | Thử nghiệm và tinh chỉnh prompt cho các vai |
| GitHub Copilot | Hỗ trợ viết code khi chỉnh sửa thủ công |

---

## Giao diện được sinh bởi AI

### index.html + style.css + app.js
- **Công cụ**: Replit Agent
- **Mô tả yêu cầu ban đầu**: *"Tạo một trang web phát hiện lừa đảo Zalo/tin nhắn, dùng OpenAI streaming, có vai thám tử + tâm lý, highlight từ khoá nguy hiểm, đồng hồ rủi ro, lịch sử, dark/light mode"*
- **Số vòng lặp chỉnh sửa**: ~8 vòng
- **Thay đổi thủ công sau khi AI sinh**: Tinh chỉnh màu sắc, sắp xếp lại cấu trúc thư mục
